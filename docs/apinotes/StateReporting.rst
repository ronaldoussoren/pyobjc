API Notes: StateReporting framework
====================================

The full API is described in `Apple's documentation`__, both
the C and Objective-C APIs are available (but see the `API Notes`_ below).

.. __: https://developer.apple.com/documentation/statereporting/?preferredLanguage=occ

These bindings are accessed through the ``StateReporting`` package (that is, ``import StateReporting``).
